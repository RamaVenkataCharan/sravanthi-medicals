# 🏥 Sravanthi Medicals - Pharmacy Billing System

A production-ready full-stack pharmacy billing and inventory management system.

## 🚀 Tech Stack
- Backend: FastAPI, SQLAlchemy, SQLite
- Frontend: React (Vite)
- Mobile (In Progress): Flutter
- Search: RapidFuzz (Fuzzy Search)

## ✨ Features
- GST-compliant billing system
- Batch-wise inventory tracking
- Expiry validation and stock control
- Fuzzy medicine search (like Google)
- Invoice generation (print-ready)
- Role-based access (Admin/Staff)
- Real-time billing and stock updates

## 📱 Upcoming
- Flutter mobile app
- Barcode scanning
- Cloud deployment
